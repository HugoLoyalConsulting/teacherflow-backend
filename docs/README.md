# Docs Hub

Ponto de entrada da documentacao do TeacherFlow.

## Mapa Rapido

- `docs/overview/`: onboarding de leitura e guias introdutorios
- `docs/deployment/`: deploy, setup e checklist operacional
- `docs/architecture/`: arquitetura e integracao de stack
- `docs/operations/`: runbooks e operacao recorrente
- `docs/security/`: seguranca, hardening e politicas
- `docs/qa/`: QA, testes e validacoes
- `docs/status/`: diagnosticos e status executivo
